# About CARGC on the Globe

## What is it?

This map charts sites around the globe that are important to the CARGC community. It includes research sites, archives, universities, institutions, the center's global collaborative projects, and sites of inspiration for our fellows and staff members. The map's framework was designed by 2024-26 Postdoctoral Fellow **Ennuri Jo** as part of her digital humanities project, Aqueous Earth Catalog. It is an open source framework intended to be used as a starting point for building a web application geared towards exploring Media throughout time and space. Find out how to build your own media map here: [GitHub repository](https://github.com/Aqueous-Earth-Catalog-Team/media-mapper).

The framework for this project was developed by Ben Tyler and [Lost Creek Designs](https://lostcreekdesigns.co/).
