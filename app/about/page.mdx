# About Turning Points: Lockdown Aftermaths

## What is it?

This map charts mark media objects that demonstrated and/or were created during the COVID-19 Lockdowns around the globe. It includes advertisements for PPE, news articles about the pandemic and the various methods of intervention, and objects that mark the afterlives of the pandemic. The map is now archived; it was created as an aid for a Turning Points workshop series at the Center for Advanced Research in Global Communications during the Spring semester of 2026. The map's framework was designed by 2024-26 Postdoctoral Fellow **Ennuri Jo** as part of her digital humanities project, Aqueous Earth Catalog. It is an open source framework intended to be used as a starting point for building a web application geared towards exploring Media throughout time and space. Find out how to build your own media map here: [GitHub repository](https://github.com/Aqueous-Earth-Catalog-Team/media-mapper).

The framework for this project was developed by Ben Tyler and [Lost Creek Designs](https://lostcreekdesigns.co/).
